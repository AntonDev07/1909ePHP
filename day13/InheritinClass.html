Kế thừa trong class JS
        để 1 class kế thừa từ 1 class ta dùng từ khóa extends
        ví dụ : classA extends classB
        Trong constructor của class con nếu muốn gọi đến constructor
        của class cha ta dùng từ khóa super()
        Bên trong class con bạn có thể gọi đến các thuộc tính và
        phương thức của class cha bằng từ khóa this
